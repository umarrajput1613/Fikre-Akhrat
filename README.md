# 🌙 Fikre-Akhrat | فکرِ آخرت

> *“Reviving hearts through Ahadith, Islamic reminders, and the remembrance of Akhirah.”*  

---

## 📖 About The Project

**Fikre-Akhrat** is a modern Islamic website created to spread the message of Islam through:

- Authentic Ahadith
- Quranic reminders
- Islamic teachings
- Daily Naseehah
- Remembrance of the Hereafter (Akhirah)

The project focuses on providing a beautiful, peaceful, and spiritually inspiring user experience with an elegant Islamic UI design.
